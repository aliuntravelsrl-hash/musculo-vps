- **Decisión Ejecutiva**: Se aprueba avanzar con _Antigravity v2.0 + Laboratorio ComfyUI_.
- **Límites**: ComfyUI NO reemplaza n8n como router principal. Su core source-code de GitHub NO será modificado. No configurar publicación automática interconectada a redes en esta fase inicial.
- **Enfoque Servidores**: Despliegue primario tipo _"Cloud-first, Local-lab"_. Probar local solo hasta entender el workflow base y saltar directo a proveedor Cloud por segundos (Ej: RunPod). Probar con modalidad vertical 9:16 hotelera como única misión activa.

### 102 - Cognición Perenne en GitHub
- **Decisión Ejecutiva**: Todo el conocimiento sistémico se volcará a directorios `docs/` en GitHub en sintáxis Markdown y JSON, y permanecerá consolidado en el repo primario (musculo-vps).
